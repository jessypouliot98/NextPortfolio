# Next Portfolio
My Next.js portfolio app

## Setup
1. copy `.env.example` to create `.env`
2. fill in the missing properties from `.env`

## Running locally
1. `yarn install`
2. `yarn dev`

## Deploying manually
1. SSH into the server
2. go into the project directory on the remote server
3. run the deploy script
   (`sh ./deploy.sh`)
